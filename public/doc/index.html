post /auth/login

Login to the device and get a token.

  • Request
  • Response

Body

Type: application/json

{
                                    	"phone": 111111111,
                                    	"password":"password",
                                    	"device": "deviceuniquekey"
                                    }

Responses

  • 200: Successfully logged-in
  • 400: An exception occurred while logging in
  • 422: Validation error
